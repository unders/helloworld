# Helloworld
A web server that replies: Hello world

# Usage

Type in console 1:
````
$ go get github.com/unders/helloworld
$ helloworld

````

Type in console 2:
````
$ curl localhost:8080
Hello world
````


You can also start it with another address:
````
helloworld 0.0.0.0:80
````
